# Projet-dema1n.org

### Init le projet

- Commence par le cloner. Une fois dans le répertoire cible, utilise la commande 

```
npm i
```

Pour installer les dépendances.

Utilise la commande
```
npm start
```
pour démarrer le projet par défaut sur le port 3000 de ton localhost.

http://localhost:3000/

J'ai placé quelque bout de code histoire de te familiariser avec [ReactJS](https://react.dev/learn).
